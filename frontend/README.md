# SAHARA – Smart AI Healthcare Assistance & Rapid Aid

A modern AI-powered healthcare web application for Attock, Punjab, Pakistan.

## 🚀 How to Run

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Open in browser
```
http://localhost:5173
```

## 📁 Project Structure
```
sahara/
├── src/
│   ├── components/
│   │   ├── Navbar.jsx       # Sticky navigation bar
│   │   ├── Footer.jsx       # Footer with emergency contacts
│   │   └── Chatbot.jsx      # Floating AI chatbot (SARA)
│   ├── pages/
│   │   ├── Home.jsx         # Hero, slideshow, features
│   │   ├── AICheck.jsx      # 4-step AI symptom checker
│   │   ├── Doctors.jsx      # Doctor directory with filter
│   │   ├── Hospitals.jsx    # Hospitals with Leaflet map
│   │   ├── Emergency.jsx    # Emergency GPS navigation
│   │   └── About.jsx        # About SAHARA project
│   ├── data/
│   │   └── data.js          # Doctors, hospitals, symptom data
│   ├── App.jsx              # Main router
│   ├── main.jsx
│   └── index.css            # Global styles + animations
├── index.html
├── package.json
├── tailwind.config.js
├── vite.config.js
└── postcss.config.js
```

## ✅ Features
- 🤖 AI Symptom Checker (4-step flow)
- 🚨 Emergency GPS Navigation (Leaflet + OpenStreetMap)
- 👨‍⚕️ Doctor Directory with filtering
- 🏥 Hospital Registry with map
- 💬 Floating AI Chatbot (SARA)
- 📱 Fully Responsive
- 🎨 Glassmorphism + Gradient Design

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS
- React Router
- Leaflet.js + React-Leaflet
- OpenStreetMap
